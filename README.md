# Iron Pulse Gym | Web Project 🏋️‍♂️

Welcome to my first official web project! This is a modern landing page for a fitness center I named **Iron Pulse**. 

 🎯 Project Goal
The main goal of this project was to take everything I learned in **HTML, CSS, and JavaScript** and build a real-world, functional website. I wanted to create a page that looks great, feels energetic, and works perfectly on every device.

 What’s Inside?
1. **Fully Responsive Design:** This was my biggest focus. I made sure the website "shapes itself" depending on the screen size. If you open it on a phone, the horizontal links disappear and a sleek **Hamburger Menu** (3-bar icon) appears to keep the design clean.
2. **Modern UI/UX:** I chose bold colors and high-energy images to match the gym atmosphere. I wanted the user to feel motivated as soon as they landed on the page.
3. **Clear Sections:** The site is organized into easy-to-navigate parts:
   * **Hero Section:** A powerful first impression.
   * **Programs:** Showcasing Cardio, Strength Training, and Yoga.
   * **Membership:** Clear pricing plans for potential members.
4. **Registration Page:** I created a separate page for users to sign up and join the gym.

🛠️ Tools I Used
* **HTML5:** To build the solid structure of the page.
* **CSS3:** For styling, layout (Flexbox/Grid), and making the site responsive.
* **JavaScript:** To add interactivity, specifically making the mobile menu toggle on and off.
* **VS Code & Git:** My workspace for writing code and pushing it to GitHub.

🌟 Conclusion
I’m really proud of how this project turned out as my first step into web development. It taught me how to solve problems and think like a developer. I'm excited to keep adding more features and improving my skills!
